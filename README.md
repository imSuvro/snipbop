# SnipBop

SnipBop is a lightweight Next.js app scaffolded with TypeScript and the App Router.

## Stack

- Next.js
- React
- TypeScript
- ESLint
- App Router

## Scripts

```bash
npm run dev
npm run lint
npm run build
npm run start
```

## Local Development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open http://localhost:3000.
